# astrbot-plugin-helloworld

《不懂 github 的我决定写 QQ Bot 插件来取悦自己？~ 如果哪天写不下去了就删库跑路吧 haha》

总而言之是实现各种自己需要的功能的 Bot。目前已实现功能：

`/savememe`: QQ 的表情包存储空间只有 300(SVIP 1000) 个？这根本不够用啊混蛋！耍我啊混蛋！
如果遇到想要好的表情包，但是点下“添加到表情”时却弹出“收藏的表情数量已到上线”……？这时，就拜托 Bot 将表情包存储到本地空间吧！
